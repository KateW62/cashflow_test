🚀 如何运行游戏

新的PC端优化版本已创建完成！按照以下步骤启动游戏：

🎮 启动开发服务器 📂 查看源码

命令行操作：

cd cashflow_test

# 安装依赖（如需要）

# 启动开发服务器
npm run dev

# 或者使用我们创建的启动脚本
./start_electric_velocity.sh
